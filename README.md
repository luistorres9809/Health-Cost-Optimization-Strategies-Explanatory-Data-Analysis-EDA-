# Health Cost Optimization Strategies — Exploratory Data Analysis (EDA)

## Business Context
HealthWise (private medical clinic) faces financial pressure due to inefficient cost management and limited visibility into service quality. This project performs exploratory data analysis to identify patterns and factors associated with healthcare costs and operational efficiency.

## Objective
- Understand cost drivers
- Identify opportunities to reduce unnecessary spending
- Support evidence-based decisions through data insights

## Tech Stack
- Python
- Pandas / NumPy
- Matplotlib / Seaborn
- Jupyter Notebook


## Analysis Workflow
1. Data loading and cleaning
2. Exploratory analysis (distributions, correlations, segmentations)
3. Visualization of cost drivers and trends
4. Insights and recommendations
